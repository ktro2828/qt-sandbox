# watch-app
An app of display watch

## Build

### Requirements

- Qt5

```shell
# check your Qt version
$ qmake --version
QMake version 3.1
Using Qt version 5.15.3 in /usr/lib/x86_64-linux-gnu
```

```shell
$ cmake -B build
$ cmake --build build
```
